/**************************************************** * ESP32 + MQTT + DHT11 * - Controle do rel√©: t√≥pico "casasenai/rele" * - Envio de temperatura: t√≥pico "senai/temperatura" ****************************************************/ #include #include #include "DHT.h" // biblioteca do sensor DHT // -------- CONFIGURA√á√ïES DE REDE -------- const char* ssid = "ReMoTe"; const char* password = "remote2025"; // -------- CONFIGURA√á√ïES DO MQTT -------- const char* mqttServer = "broker.emqx.io"; const int mqttPort = 1883; const char* topicRele = "senaimesa6/rele"; // t√≥pico de controle const char* topicTemp = "senaimesa6/temperatura"; // t√≥pico de temperatura // -------- HARDWARE -------- #define DHTPIN 26 // pino de dados do DHT11 #define DHTTYPE DHT11 // tipo do sensor const int rele = 27; // pino do rel√© DHT dht(DHTPIN, DHTTYPE); // cria objeto DHT WiFiClient espClient; PubSubClient client(espClient); unsigned long ultimoEnvio = 0; // controle de tempo // -------- CALLBACK (mensagens recebidas) -------- void callback(char* topic, byte* payload, unsigned int length) { String msg = ""; for (unsigned int i = 0; i < length; i++) msg += (char)payload[i]; Serial.printf("[MQTT] %s => %s\n", topic, msg.c_str()); if (msg == "on") { digitalWrite(rele, HIGH); Serial.println("Rel√© LIGADO"); } else if (msg == "off") { digitalWrite(rele, LOW); Serial.println("Rel√© DESLIGADO"); } } // -------- CONECTA AO WI-FI -------- void conectaWiFi() { Serial.printf("Conectando ao Wi-Fi %s", ssid); WiFi.begin(ssid, password); while (WiFi.status() != WL_CONNECTED) { delay(500); Serial.print("."); } Serial.printf("\nWi-Fi conectado! IP: %s\n", WiFi.localIP().toString().c_str()); } // -------- CONECTA AO BROKER MQTT -------- void conectaMQTT() { while (!client.connected()) { Serial.print("Conectando ao broker MQTT... "); String clientId = "ESP32_" + String(random(0xffff), HEX); if (client.connect(clientId.c_str())) { Serial.println("Conectado!"); client.subscribe(topicRele); Serial.print("Inscrito em: "); Serial.println(topicRele); } else { Serial.print("Falha (rc="); Serial.print(client.state()); Serial.println("). Tentando em 3s..."); delay(3000); } } } // -------- SETUP -------- void setup() { delay(2000); Serial.begin(115200); Serial.println("Iniciando..."); pinMode(rele, OUTPUT); digitalWrite(rele, LOW); dht.begin(); // inicializa sensor DHT randomSeed(micros()); conectaWiFi(); client.setServer(mqttServer, mqttPort); client.setCallback(callback); } // -------- LOOP -------- void loop() { if (WiFi.status() != WL_CONNECTED) conectaWiFi(); if (!client.connected()) conectaMQTT(); client.loop(); // l√™ o DHT a cada 10 segundos if (millis() - ultimoEnvio > 10000) { ultimoEnvio = millis(); float h = dht.readHumidity(); float t = dht.readTemperature(); if (!isnan(h) && !isnan(t)) { char payload[50]; snprintf(payload, sizeof(payload), "{\"temp\":%.1f,\"umid\":%.1f}", t, h); client.publish(topicTemp, payload); Serial.printf("üì§ Enviado: %s\n", payload); } else { Serial.println("‚ùå Falha ao ler o DHT11"); } } }
